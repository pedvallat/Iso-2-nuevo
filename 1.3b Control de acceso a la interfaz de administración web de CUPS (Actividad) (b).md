1.3b Control de acceso a la interfaz de administración web de CUPS (Actividad) (b)

ARCHIVO PRINCIPAL A MODIFICAR:
sudo nano /etc/cups/cupsd.conf

1- PERMITIR ACCESO A LA INTERFAZ WEB DE CUPS DESDE CUALQUIER ORDENADOR DE LA RED

Se modifica en cupsd.conf este bloque y debe de quedar asi:
<Location /admin>
  Order allow,deny
  Allow from all
  AuthType Default
  Require valid-user
</Location>

2- HABILITAR SOLO ACCESO LOCAL O DESDE UNA IP ESPECÍFICA

2.1 Acceso solo local se debe dejar la configuración asi:
<Location />
  Order allow,deny
  Allow from 127.0.0.1
  Allow from ::1
</Location>

2.2 Acceso desde una IP por ejemplo la 192.168.56.11 debe quedar asi:
<Location />
  Order allow,deny
  Allow from 192.168.56.11
</Location>

<Location /admin>
  Order allow,deny
  Allow from 192.168.56.11
  AuthType Default
  Require valid-user
</Location>

Solo ese equipo podra ver la interfaz.

3- PERMISOS AVANZADOS A USUARIOS Y GRUPOS

3.1 Acceso total a la administración para pepet en la sección /admin :
<Location /admin>
  AuthType Default
  Require user pepet
  Order allow,deny
  Allow from 192.168.56.0/24
</Location>

Ahora solo pepet puede acceder al menú completo.

3.2 Grupo profesores: administrar trabajos de impresión en la sección /admin/jobs :
<Location /admin/jobs>
  AuthType Default
  Require group profesores
  Order allow,deny
  Allow from 192.168.56.0/24
</Location>

3.3 Cualquier usuario autenticado administrar impresoras en la seccion /admin/printers :
<Location /admin/printers>
  AuthType Default
  Require valid-user
  Order allow,deny
  Allow from 192.168.56.0/24
</Location>
